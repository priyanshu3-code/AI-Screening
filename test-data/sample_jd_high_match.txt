JOB DESCRIPTION

POSITION: Senior Full-Stack Software Engineer
COMPANY: TechVision Inc.
LOCATION: San Francisco, CA (Hybrid)
EMPLOYMENT TYPE: Full-Time
SALARY RANGE: $160,000 - $220,000 + Benefits

ABOUT US
TechVision Inc. is a rapidly growing SaaS company serving enterprise clients in the financial services and healthcare sectors. We're building next-generation cloud-native applications that process millions of transactions daily. Our engineering team is committed to excellence, scalability, and continuous innovation.

ROLE OVERVIEW
We're seeking a Senior Full-Stack Software Engineer to join our Platform Engineering team. You'll take ownership of critical backend services and contribute to our modern Angular-based frontend. You'll work in a collaborative environment where your technical expertise, architectural thinking, and mentorship of junior engineers directly impact our product's success.

KEY RESPONSIBILITIES
• Design and develop microservices using Spring Boot 3.x handling high-volume API traffic
• Architect RESTful APIs with clear documentation (OpenAPI/Swagger)
• Implement robust authentication, authorization, and rate-limiting mechanisms
• Optimize database performance (PostgreSQL/MySQL) for large-scale data operations
• Build responsive, user-friendly interfaces using Angular 18 and modern frontend frameworks
• Implement comprehensive testing strategies (JUnit 5, Mockito) with >80% code coverage
• Deploy applications using Docker containers and Kubernetes orchestration
• Mentor junior engineers and conduct thorough code reviews
• Participate in on-call rotation for production support
• Collaborate with product, design, and DevOps teams to deliver high-quality solutions

REQUIRED QUALIFICATIONS
• 7+ years of software development experience with 4+ years in backend development
• Expert-level proficiency in Java 21, Spring Boot 3.x, and Spring Data JPA
• Strong experience building and consuming RESTful APIs at scale
• Proven expertise with PostgreSQL or similar relational databases
• Experience with containerization (Docker) and orchestration (Kubernetes)
• Proficiency in Angular 18 (or equivalent modern frontend framework)
• Solid understanding of microservices architecture and design patterns
• Experience with CI/CD pipelines (GitHub Actions, Jenkins, or similar)
• Strong testing practices: JUnit 5, Mockito, integration testing
• Bachelor's degree in Computer Science or equivalent professional experience
